Parámetro Valor Descripción
Este parámetro se refiere al modelo de lenguaje natural que se utilizará para generar la respuesta. Los modelos de lenguaje natural se entrenan en grandes conjuntos de datos de texto para aprender patrones y estructuras del lenguaje. El modelo que se utiliza debe ser específico para la tarea que se está abordando, por ejemplo, responder preguntas o generar texto creativo.
Este parámetro controla el grado de aleatoriedad en la generación de respuestas. Un valor alto de temperatura produce respuestas más variadas y creativas, pero también puede generar respuestas menos coherentes. Un valor bajo de temperatura genera respuestas más predecibles y coherentes, pero también más limitadas.
Este parámetro establece el número máximo de palabras que el modelo puede generar en una respuesta. Si se establece un valor bajo, la respuesta será más corta y puede ser más relevante. Si se establece un valor alto, la respuesta puede ser más detallada, pero también puede ser menos precisa. (Valor máximo 2048)
Este parámetro controla la cantidad de opciones que el modelo considera al generar la respuesta. El modelo generará una lista de posibles palabras para cada posición en el texto, y luego seleccionará las palabras con la probabilidad acumulada más alta que sumen el valor establecido en top_p.
Este parámetro penaliza las palabras que el modelo ha generado con una frecuencia más alta en el conjunto de datos de entrenamiento. La idea es que el modelo genere palabras más creativas y menos comunes al penalizar las palabras que usa con más frecuencia.
Este parámetro penaliza las palabras que el modelo ya ha generado en la respuesta. La idea es que el modelo genere respuestas más variadas y evite repetir las mismas palabras o frases.
El número de respuestas que se generarán para la entrada proporcionada.
Una lista de palabras o frases que se utilizarán para detener la generación de la respuesta. Esto puede ser útil para limitar la longitud de la respuesta o para evitar que el modelo genere información irrelevante.
